Invoice System - C#

This project, completed for CS 3280 - Object Oriented Windows, was a group assignment to develop a Windows WPF application simulating an invoice system for a fictional business. Each team member was responsible for a different page; I worked on the Main page, which handled creating new invoices, adding items, viewing older invoices, and submitting the completed invoice.

Text Prediction - Jupyter Notebook

This project, completed for CS 4580 - Data Science & Algorithms, served as an introduction to text-based identification using predictive algorithms. We analyzed the writing styles of multiple authors, using Principal Component Analysis (PCA) to visualize stylistic similarities. Various prediction models were then applied to evaluate how accurately they could identify the author of a new, added book. The project was divided into two parts: a guided first phase and a more independent second phase.

Android Superhero API App

This project, completed for CS 3270 - Mobile Develop for Android, was a beginner-level Android application. The app allows users to select two superheroes using data retrieved from an external API. It features a functionality that can randomly select two superheroes and compare their stats to determine a hypothetical winner in a head-to-head battle. While simple in design, the app serves as an introduction to core Android development concepts such as API integration, UI design, and basic logic implementation.
